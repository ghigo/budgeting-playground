How Auto-Categorization Works

Transactions are categorized automatically using a 4-step hybrid approach:

  1. Merchant Lookup: Exact and fuzzy matching against known merchants
  2. Pattern Rules: Regex patterns for common merchant types (Walmart, Amazon, gas stations, etc.)
  3. Plaid Categories: Mapping Plaid's built-in categories to your categories
  4. Manual: Uncategorized transactions can be manually assigned

üí° The system learns from your manual categorizations and builds the merchant mappings automatically!
